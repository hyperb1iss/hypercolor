# hypercolor-leptos-ext

In-tree canary for the Cinder-inspired Leptos and stream primitives.

- Design spec: `docs/design/43-hypercolor-leptos-ext-spec.md`
- Audit snapshot: `docs/design/cinder-audit-snapshot.txt`

This crate starts as a buildable scaffold with strict feature gates so Hypercolor
can verify native workspace builds and WASM UI builds before larger migrations.
