live tint demos 3 flavorssection · 01

All three render at runtime, using only the masks under mask/. Try the color pickers; click rainbow cycle to see each petal phase 120° apart on the hue wheel.

flavor 01

single wash

Whole trinity tints to one color. Simplest. Drives off average_frame_color.

flavor 02

per-petal (tri)

Three primaries, three independent zones. The killer feature.

flavor 03

chrome + tint

Original 3D chrome detail preserved, hue shifts with scene. Prettiest.
